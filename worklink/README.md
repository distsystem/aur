# worklink

招商随行（[WorkLink](https://www.cm-worklink.com)）的 Arch Linux 打包。

> 已迁移至 AUR，本仓库不再更新。
>
> AUR 包：https://aur.archlinux.org/packages/worklink

## 安装

任选一个 AUR helper：

```bash
yay -S worklink
# 或
paru -S worklink
```

裸 `makepkg`：

```bash
git clone https://aur.archlinux.org/worklink.git
cd worklink
makepkg -si
```
